Entête d'un document HTML

L'entête d'un document HTML est définit par une balise <head>
à l'intérieur de laquelle est définit des éléments et leur attributs :

  • <meta><charset="UTF-8">
  • <title>
  • <link>
  • <script>
Exemple de lignes de codes
							
		<head>
			<meta charset="utf-8">
			<title> Un titre </title>
			<script src="url js">  </script>
		</head>

Le corps du document HTML

Le corps d'un document HTML est définit par la balise <body> et contient l'ensemble du contenu visible de la page web. Les balises à l'intérieur du corps sont organisées de manière hiérarchique pour structurer le contenu de la page de manière logique. On peut citer comme balises principales :

  • <header>
  • <main>
  • <footer>
  • <nav>
  • <section>
  • <h1>

Ces éléments sont généralement accompagnés par les attributs class et id pour appliquer des styles ou référencer des éléments via du JavaScript.

Exemple de corps d'un document HTML
							
		<body>
			<header>
				<h1> Ceci est un titre </h1>
			</header>
			<nav>
				<p> Ceci est paragraphe dans une balise nav </p>
			</nav>
			<main>
				<p> Ceci est un paragraphe </p>
			</main>
			<footer>
				<p> Ceci est un paragraphe dans une balise footer </p>
			</footer>
		</body>
